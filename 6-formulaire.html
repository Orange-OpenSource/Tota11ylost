Description

Les formulaires sont omniprésents sur le Web.

Il est donc impossible de s'en passer et doit être facilement utilisable pour tous.

Vous allez ici essayer de remplir correctement ce formulaire d'inscription.

Le type d'utilisateur que vous incarnez

Vous êtes dans la peau de quelqu'un souffrant d'handicap cognitif ou visuelle, donc souvent confrontées à des formulaires pas ou peu accessible.

Règles

  • Remplissez tous les champs de formulaires (vous pouvez mettre des données fictives !!!).
  • puis appuyez sur le bouton "s'incrire".
  • Une fois le formulaire validé, il vous sera possible de valider l'énigme.
  • Si c'est trop dur, n'hésitez pas à prendre des indices !!!
  • Attention, les indices rajoutent du temps à votre timer...
