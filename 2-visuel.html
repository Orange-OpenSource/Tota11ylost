Handicap visuel

Fin du temps imparti

Description

La navigation sur internet est un défi pour une personne atteinte de déficience visuelle.

L'expérience en ligne est très dépendante des aides techniques (comme les lecteurs d'écran) permettant de compenser en partie la situation de handicap, ET (additif !) d'un code HTML respectant les normes d'accessibilité numériques (WCAG) pour s'interfacer correctement aux outils.

Le type d'utilisateur que vous incarnez

Vous êtes dans la peau d'un daltonien (gravement atteint). Vous ne voyez donc aucune couleur ou presque ... donc lorsque l'information n’est portée que par la couleur comme dans le cas des 4 boutons de bas de page, elle ne vous est pas accessible.

Règles

  • Cliquez sur les boutons dans un ordre précis : vert, bleu, rouge et ensuite violet pour passer cette étape
  • Attention, en cas d'erreur il faudra tout recommencer !
